# Keele Library Map – Assessment 1 (CSC-44102)
> **Private repository. Do not publish publicly.**

Two apps for the same task:
- text-based console (ui_console/cli.py)
- Tkinter GUI (ui_tk/gui.py)

Both read two CSVs from data/ and support search by: subject, classmark, location.

## Dev quick start
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
